# Hotel CRM (Radisson Blu Nagpur)

Internal Hotel CRM built on Rails with PostgreSQL, Sidekiq, Redis, Bootstrap 5, and RSpec.

## Documentation
- Client and operations guide: `docs/CLIENT_PROJECT_DOCUMENTATION.md`

## Quick Start
```bash
bundle install
yarn install
bin/rails db:setup
bin/dev
```

## Run Tests
```bash
bundle exec rspec
```
